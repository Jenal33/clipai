Architecture & Stack

🏗️ Tech Stack
// full-stack overview
Frontend Next.js 14+ (App Router) + Tailwind CSS
Backend API FastAPI (Python) atau Fastify Node.js
Queue / Workers BullMQ + Redis antrian proses video
Storage Cloudflare R2 atau AWS S3
Database PostgreSQL + Prisma ORM
Auth NextAuth.js atau Supabase Auth
Payment Midtrans pasar Indonesia
Deployment Vercel + Railway atau Render.com
⚙️ Processing Pipeline
// video processing flow
01 Download Video yt-dlp --format best
02 Extract Audio ffmpeg -vn -acodec pcm
03 Transkripsi whisper large-v3 --lang id
04 Analisis Viral Claude API / GPT-4o
05 Face Detection mediapipe.solutions.face
06 Render 9:16 ffmpeg -vf crop+scale
07 Burn Captions ffmpeg -vf subtitles=
08 Virality Score custom_scorer.predict()
09 Upload ke Storage r2.put_object(key, data)

💳 Payment Flow — Midtrans Token Top-Up
// sequence diagram
User klik "Top Up 500 Token - Rp 49K" ↓ POST /api/payment/create-transaction → Buat Midtrans Snap transaction (order_id, amount, metadata: {userId, tokenAmount}) ← Return snap_token ↓ Frontend buka Midtrans Snap popup → User bayar via QRIS / Transfer / VA / GoPay / OVO ↓ POST /api/payment/webhook (Midtrans callback) → Verifikasi signature key → Jika transaction_status == 'settlement': → db.user.update({ tokenBalance: { increment: tokenAmount } }) → Kirim notif ke user ↓ Token otomatis masuk, user bisa langsung generate klip
